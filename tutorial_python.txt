O venv é uma ferramenta em Python que permite criar ambientes virtuais isolados para cada projeto.
 Isso é útil para manter as dependências de um projeto separadas de outros, evitando conflitos entre pacotes e facilitando a distribuição do projeto.
  Quando você ativa um ambiente virtual, o Python e todas as bibliotecas instaladas ficam confinadas dentro desse ambiente, o que ajuda a manter a consistência e a organização do desenvolvimento.
